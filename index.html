SVG Sprites Playground

Playground to test out different polyfills and use cases for SVG sprites including React.

Regular HTML

<svg class="icon">
  <use xlink:href="assets/sprite.svg#burger"></use>
</svg>

React JSX

ReactDOM.render(
  <svg className="icon">
    <use xlinkHref="assets/sprite.svg#arrow-up" />
  </svg>
)

React on Trigger

React Stateful

React Stateful with ReactSprite

class Icon extends Component {
  render() {
    return (
      <ReactSprite
        href={`assets/sprite.svg#${this.state.icon}`}
      />
    )
  }
}
